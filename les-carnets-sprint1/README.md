# Tout part de toi · Les Carnets

Sprint 1 : socle Astro, page d’accueil, premier Carnet, identité graphique responsive, RSS et déploiement GitHub Pages.

## Lancer en local

```bash
npm install
npm run dev
```

## Ajouter un article

Dupliquer un fichier dans `src/content/articles/`, modifier les métadonnées et écrire le contenu en Markdown.

```yaml
---
title: "Titre"
description: "Résumé"
publishedAt: 2026-07-16
category: "Neuroatypies"
tags: ["TDAH"]
image: "/images/illustration.webp"
draft: false
---
```
